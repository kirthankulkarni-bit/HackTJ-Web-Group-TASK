    Metric definitions

    Shock magnitude — Expected peak impact on equity returns (%). Negative = decline; positive = rally. E.g. -30% means a ~30% drop at the shock's peak.

    Shock duration — How many days the stress stays concentrated. Longer = slower decay; shorter = sharp spike then fast recovery.

    Volatility — Multiplier for day-to-day noise (like VIX). Higher = more erratic daily moves around the trend.

        Methodology and reference datasets

        This simulator uses a synthetic stochastic model. Parameters are inspired by:

        • St. Louis Fed Financial Stress Index — quantification of stress (FRED)
        • CBOE VIX — volatility calibration (Chicago Board Options Exchange via FRED)
        • S&P 500 — equity index behavior (S&P Dow Jones via FRED)

        No real market data is used. For retraining with real data, see train_news_model.py and train_market_model.py.
